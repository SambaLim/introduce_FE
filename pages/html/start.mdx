# HTML

> HyperText Markup Language

## HTML?

웹 페이지를 구성하는 마크업 언어입니다. HTML은 꺽쇠기호(`<`,`>`)로 둘러쌓인 태그로 되어있는 HTML Element로 구성되어있습니다.

## 기본 구조

```html
<!DOCTYPE html>
<html>
<head>
  <title>My First HTML</title>
</head>
<body>
  Hello, World!
</body>
</html>
```

### 태그

HTML에서는 마크업 방식으로 각 영역을 표시하고 HTML 문서를 구성합니다.
<br/>마크업을 할 때 사용하는 약속된 표기법을 태그라고 하는데 이는 HTML5 웹 표준을 따라야합니다.

1. 태그는 꺽쇠기호(`<`,`>`)로 이루어집니다.
2. 태그는 대, 소문자를 가리지 않습니다.

## Element

```html
<p>Hello, World!</p>
```
- 여는 태그(`<p>`): Element가 시작되는 부분으로 요소의 효과가 적용되기 시작합니다.
- 닫는 태그(`</p>`): `/`가 있는 것이 특징으로 요소의 범위를 정해줍니다.
- 내용(`Hello, World!`)
- Element(`<p>Hello, World!</p>`): 여는 태그부터 닫는 태그까지를 통틀어 Element라고 합니다.

### 속성

```html
<a href="https://www.naver.com" target="_blank">네이버로 이동하기</a>
```

Element는 웹문서내에는 보여주고 싶지 않지만, 추가적인 내용을 담고 싶을때 속성을 사용합니다.
