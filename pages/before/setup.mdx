# 설정하기

> 제가 사용하고 있는 에디터, 브라우저를 소개하고 HTML 문서를 실행하는 과정을 함께합니다.

## Chrome

1. 구글에 'Chrome'을 검색하신 후, [Google Chrome](https://www.google.co.kr/intl/ko/chrome/) 페이지로 이동합니다.
2. [Chrome 다운로드]버튼을 사용하여 쉽게 다운로드하실 수 있습니다.

## VS Code

> 메모장부터 시작해서 Notepad++, Webstorm등 다양한 에디터가 있지만, 저는 VS Code를 사용하고 있어 소개해드립니다.

1. 구글에 'VS Code'를 검색하신 후, [Visual Studio Code > Download](https://code.visualstudio.com/download) 페이지로 이동합니다.
2. 사용하고 계신 OS에 맞추어 다운로드합니다.

### Extensions

> VS Code에서 편하게 코드를 작성하기 위해 Extensions를 설치합니다. VS Code 좌측 메뉴중 Extensions를 눌러 시작하실 수 있습니다.

- HTML Snippets

## HTML 문서 시작하기

> HTML, CSS, JS 를 모두 사용하는 HTML 페이지를 만들어보려 합니다.

### HTML

1. 원하는 디렉토리에 `web`폴더를 생성합니다.
2. `web`폴더 내에 `index.html`, `index.css`, `index.js` 파일을 생성합니다.
3. `index.html` 파일 내에 아래의 코드를 입력합니다.

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta http-equiv="X-UA-Compatible" content="IE=edge">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <link rel="stylesheet" href="./index.css">
  <title>FE Class</title>
</head>
<body>
  <div>Hello, FE Class!</div>
  <script src="./index.js"></script>
</body>
</html>
```

### CSS

1. `index.css` 파일 내에 아래의 코드를 입력합니다.

```css
div {
  text-align: center;
  font-size: 32px;
  color: cornflowerblue;
}
```

### JS

1. `index.js` 파일 내에 아래의 코드를 입력합니다.

```javascript
alert('Hello, FE Class!');
```

HTML, CSS, JS 파일을 생성하고 위의 코드를 모두 입력하였다면 브라우저(Chrome)를 통해 `index.html` 파일을 실행해봅니다.
